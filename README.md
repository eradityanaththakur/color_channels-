Understanding and Working with Color Channels in Images
This Google Colab notebook is designed for high school students to explore the fascinating world of digital images, specifically focusing on how colors are represented and manipulated using color channels. You'll learn that every image you see on a screen is made up of different amounts of Red, Green, and Blue light!

What You'll Learn:
How digital images store color information.

What "color channels" are (Red, Green, and Blue).

How to load an image in Python using the OpenCV library.

How to split an image into its individual Red, Green, and Blue channels.

How to visualize each color channel separately.

How to Use This Notebook:
Open in Colab: Click the "Open in Colab" badge at the top of this notebook (if available) or go to Google Colab and upload this .ipynb file.

Run Cells: Go to Runtime -> Run all or click the "Play" button next to each code cell to execute it step-by-step.

Follow Instructions: Read the text explanations and comments in the code to understand what's happening.
